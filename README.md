# "StudyFlow" Web Application

## Project Description
This project is a web application that will be developed using Angular and Django REST Framework.
The main goal of the project is to help students manage their academic tasks, courses, and deadlines in an organized and efficient way.
The system will allow users to create tasks, assign them to courses, track deadlines, and monitor their progress.
